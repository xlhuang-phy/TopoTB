# TopoTB

A software package for calculating the electronic structure and topological properties of tight-binding models.

## Features

- Band structure, density of states, Fermi surface, and spin texture
- Berry phase, Berry curvature, Chern number, and Z₂ number
- VASP interface for processing band data
- Wannier90 interface for analyzing `wannier90_hr.dat` files
- Batch testing of TB model parameters
