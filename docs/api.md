# API Documentation

TopoTB includes the following modules:

| Module                            | Description                        |
| --------------------------------- | ---------------------------------- |
| `TopoTB`BrillouinZone2D`*`        | 2D Brillouin zone calculations     |
| `TopoTB`FirstBrillouinZone`*`     | Brillouin zone handling            |
| `TopoTB`HamiltonianCalculation`*` | Hamiltonian calculations           |
| `TopoTB`TopologicalNumber`*`      | Topological invariants (Chern, Z₂) |
| `TopoTB`Model`*`                  | Model construction                 |
| `TopoTB`Wannier90`*`              | Wannier90 interface                |
| `TopoTB`VASP`*`                   | VASP interface                     |
| `TopoTB`Plots`*`                  | Plotting utilities                 |

## Getting help

Use `?` followed by the function name. Example:

    ?HSPBand
